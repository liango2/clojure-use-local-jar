# clojure-use-local-jar

# 引入本地jar包:

例如 本地jar包存放在 C:\Users\liango\clojure-use-local-jar\resources目录下

在project.clj中使用
 ```
    :resource-paths ["resources/*"]
 ```
来引入本地的包:
